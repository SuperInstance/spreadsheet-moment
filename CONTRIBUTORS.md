## How to Contribute

We welcome contributions from everyone! See [CONTRIBUTING.md](CONTRIBUTING.md) for details.

### Ways to Contribute

- **Code:** Bug fixes, features, performance improvements
- **Documentation:** Guides, tutorials, API docs
- **Tests:** Unit tests, integration tests, E2E tests
- **Design:** UI/UX, themes, components
- **Integration:** Claw integration, plugins, extensions
- **Community:** Support, advocacy, outreach

### Getting Started

1. Read [CONTRIBUTING.md](CONTRIBUTING.md)
2. Check [good first issues](https://github.com/SuperInstance/spreadsheet-moment/issues?q=is%3Aissue+is%3Aopen+label%3A%22good+first+issue%22)
3. Join [Discussions](https://github.com/SuperInstance/spreadsheet-moment/discussions)
4. Start contributing!

### Recognition Levels

**Bronze Contributor** (1-4 contributions)
- Listed in contributors
- Thanked in release notes

**Silver Contributor** (5-9 contributions)
- Featured contributor status
- Invited to maintainer discussions

**Gold Contributor** (10+ contributions)
- Hall of fame listing
- Permanent recognition
- Annual contributor gift

### Awards

**Contributor of the Month**
- Highlighted in blog posts
- Featured on social media
- Special badge

**Contributor of the Year**
- Presented at annual summit
- Permanent recognition
- Custom award

## Become a Maintainer

We're always looking for new maintainers!

**Requirements:**
- Consistent, high-quality contributions
- Active community participation
- Technical expertise
- Endorsed by existing maintainers

**Benefits:**
- Full repository access
- Vote in decisions
- Represent the project
- Shape the future

**Process:**
1. Contribute consistently
2. Review pull requests
3. Help with releases
4. Be nominated by existing maintainers
